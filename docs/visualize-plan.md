
# 可更新共识的可视化规划文档

根据以下参数指标，制作可视化大屏页

## 可视化的指标

### 整体架构板块
1. 共识节点数量
2. 支持的共识类型
3. 共识池
 -  共识池状态：占用中/空闲中
4. 网络服务状态和利用率：正常/百分比
5. 执行器状态：正常/异常
6. 存储空间消耗
7. 交易池大小
8. 交易吞吐率
    
### 分共识状态
1. POT
- 当前epoch数量
- 挖矿难度值
- 当前区块高度
- VDF0迭代进度
- VDF1状态

### 委员会共识
- 共识类型 (Simple Whirly / CR Whirly / Whirly)
- 委员会大小 (当前: 4, 参数: Commiteelen)
- 委员会数量 (SmallN: 4)
- 确认延迟 (ConfirmDelay: 6)
- 委员会工作高度
- 批处理大小 (BatchSize: 2-10)
- 委员会激活状态 (inCommittee: true/false)
- 委员会权重 (Weight)

### 委员会密码学参数
- DPVSS恢复门限 (Threshold)
- 候选密钥长度 (CandidateKeyLen: 32)
- 本地SRS (结构化参考字符串)
- 置换后的公钥列表
- 抽签随机数承诺
- 未启用的委员会队列
- 加密份额聚合状态
- 份额解密状态

### 委员会阶段状态
- 初始化阶段 (height <= 32): 参数生成
- 置换阶段 (height % 32 in [1, 4]): 公钥置换
- 抽签阶段 (height > 36): 委员会选举
- DPVSS阶段 (height > 37): 分布式份额分发
- 工作阶段 (height > BigN + 2*SmallN): 委员会执行共识

### 激励状态
1. 总奖励金额
2. 锁定的激励金额
3. 利息总数
4. 奖励分配比例
5. 锁定利率参数
