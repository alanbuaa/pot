
## Upgradeable Consensus

`upgradeable-consensus` is a simple implementation of consensus upgrade framework. 

# Run the example
## build 
```
make 
```

## generate key
```
generate_key
```

## test
```
make test
```

## run 

Open 2 terminals , run consensus server in the first with. 
```
make run_server
```

Run client in the second with.
```
make run_client
```
